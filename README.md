# 🧮 A Simple C Calculator

This is a very simple and basic calculator in C comprising of the `if-else` ladder logic in the C programming language. Also, this is my very first project in C! 🚀

## 📋 Overview
* **Functionality:** The program performs basic arithmetic operations like addition, subtraction, multiplication, and division.
* **User Experience:** The prompts in the program are user-friendly and professional.
* **Safety:** The program includes fundamental error handling for zero-division errors.

## 🚀 How to run it?
Follow the steps below, and you'll have your calculator running smooth as butter on your system!

1. **Download** the `calculator.c` file or copy-paste the code.
2. **Open** the file using VS Code or any other code editor.
3. **Compile** using the command:
   `gcc calculator.c -o calculator`
4. **Run it:** 
   `calculator.exe` (on Windows)

## 🎓 Learning Outcomes
* Application of `if-else` ladder using `if-else if-else` commands in C.
* Mastering `scanf` and `printf` commands.
* Understanding and handling the zero-division error.
